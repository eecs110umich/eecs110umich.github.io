{% comment %} Renders a list of { name, url, note } entries as a card grid. A url left as "TODO" (or blank) renders as plain text marked "coming soon" instead of a broken link, so the site is presentable before real URLs exist. Works at any list length. Usage: {% include link-cards.html items=site.data.course.links %} {% endcomment %}
    {% for item in include.items %} {% assign has_url = false %} {% if item.url and item.url != "" and item.url != "TODO" %}{% assign has_url = true %}{% endif %}
  • {% if has_url %}

    {{ item.name }}

    {% else %}

    {{ item.name }}

    {% endif %} {% if item.note and item.note != "" %}

    {{ item.note }}

    {% endif %} {% unless has_url %}

    Link coming soon

    {% endunless %}
    • {% endfor %}
