# Past-term syllabus PDFs

Drop prior-term syllabus PDFs in this folder. They're served at
`/assets/syllabi/<file>` and listed in the Home-page "Past syllabi" disclosure
via the `past_syllabi:` list in `_data/course.yml`.

Expected filenames (referenced from `course.yml`):

- `eecs110-wn2024.pdf`
- `eecs110-fa2024.pdf`
- `eecs110-wn2025.pdf`
- `eecs110-fa2025.pdf`
- `eecs110-wn2026.pdf`

To add a new term: place the PDF here and add one line to `past_syllabi:` in
`_data/course.yml`. The buttons 404 until the matching PDF is present.
